The Analyst · Pose Sheet v1

Guided-Onboarding Assets

Six poses cover every scene in the script (docs/plans/onboarding-conversion/guided-avatar-script.md). Constant identity anchors across poses: glasses, lace "hair," left/right white stripes. Bubble attaches top-center; entrance = slide-up + single overshoot bounce; idle = pupil blink every ~4s. The point pose flips horizontally (scaleX −1) for left-directed steps.

1 · Neutral

Default talking state. Used in: S0 intro, S1, S7, any un-posed line. Mouth is deadpan-flat by identity; dialogue carries the emotion.

2 · Point (foam finger)

Directs the user's tap; pupils track the target; fingertip pulses in sync with the spotlight. Flip horizontally for left targets, rotate −90° for "up." Used in: S0 field, S2 deck, S3 chip, S6 share, S7 trio.

3 · Celebrate

Win moments — happy-arc eyes, tie mid-jump, confetti. Used in: S5 re-rank reveal, S6 first like, S8 sign-off. Motion: one 8px hop with squash-and-stretch, confetti burst 400ms.

4 · Computing

Any wait state — deck generation, re-rank, server wake. Progress dots blink in sequence; bars re-randomize every 700ms. Used in: S2 pre-deck, S5 regen, cold-start mask.

5 · Thinking

Reacting to the user's input — after a pass streak, an odd tier save, hesitation (idle >10s on a step). Used in: S3 trigger line, S4 tier hints, idle nudges.

6 · Oops

Error states and honest bad news — Sleeper down, empty deck, N=0 re-rank. Glasses knocked askew, one squint, sweat drop. Used in: S0 errors, S5 zero-case, any api_request_failed while guiding.
