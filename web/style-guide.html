Do / Don't

Do

  • Ice in ≤3 places per screen; flare only on informational highlights
  • Plex Mono tabular for every data number
  • Hairline rules + surface steps for depth
  • Condensed caps for titles, sentence case for body
  • Position/tier color always paired with a text label

Don't

  • Gradients, glassmorphism, blur, neon glow
  • Emoji as icons
  • Inter / Roboto / system font stack
  • Radius > 8px (except specced pills)
  • Hover lift transforms; tinted rgba badge fills
